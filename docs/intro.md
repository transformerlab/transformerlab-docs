# Transformer Lab


Transformer Lab is a free, open-source machine learning research platform that you can run on your own computers or in the cloud.

<!-- Using Transformer Lab you can easily finetune, evaluate, export and test AI models across different inference engines and platforms. -->

## Local or Cloud, Single or Multi-user

Transformer Lab can be run in single-user mode or team mode.

If you want to run Transformer Lab just for yourself, follow the [standard install instructions here](./install/install.md).

If you want to run Transformer Lab on a team, with access to cluster of GPUs (using SLURM or Skypilot, for example) [follow these steps](./install/team.md).
