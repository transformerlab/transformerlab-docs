# Transformer Lab


Transformer Lab is a free, open-source machine learning research platform that you can run on your own computers or in the cloud. It is designed for Machine Learning Researchers as a place to initiate, track and store experiments.

<!-- Using Transformer Lab you can easily finetune, evaluate, export and test AI models across different inference engines and platforms. -->

## Local or Cloud, Single or Multi-user

Transformer Lab can be run in single-user mode or team mode. For [MacOS, Windows, and Linux setup and instal instructions click here](./install/install.md).
